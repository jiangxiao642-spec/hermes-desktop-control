## Claude Desktop 关键pitfall

**问题：** `^v{ENTER}` 合并发送时，长文本粘贴未完成就被Enter截断发送。
**根因：** ProseMirror的Ctrl+V是异步操作，SendKeys不等待粘贴完成就发Enter。
**正确做法：**
```
1. clipboard_write
2. send_keys ^v
3. UIA ValuePattern验证粘贴成功（检查文本长度）
4. send_keys {ENTER}（单独发送）
```
